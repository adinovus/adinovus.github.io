RF module with relay

 

       The RF module is a small electronic circuit used to transmit, receive, or transceive radio waves on one of a number of carrier frequencies. RF modules are widely used in consumer applications such as garage door openers, wireless alarm systems, industrial remote controls, smart sensor applications, weather monitoring system, RFID, wireless mouse technology and wireless home automation systems. They are often used instead of infrared remote controls as they have the advantage of not requiring line-of-sight operation.

 

 Features

 

* Input Power supply - 5 Volts

* Compatible for Both RF 433/ 315 Mhz

* Renesas controller Based

* Rdy to Interface upto 8 Bit Data

* Direct connect to microcontroller
