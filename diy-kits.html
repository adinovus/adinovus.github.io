PIC easy kit

     Build your own projects with pic16f877a micro controller using this full featured training kit. the kit has the provision for ICSP programming. also have the features like matrix key board , 7 segment display, ports for lcd display, rs232, i2c etc

 

features

*  ICSP connector

*  LCD connector

*  DIP switches

*  7 segment display

*  DB9 connector

*  Matrix keypad

*  real time clock ds1307

 

89v51 kit

     The kit is based on philips 89v51rd2 microcontroller.  Kit contains the boot-loader which facilitates the  user to flash the  chip simply by using rs232 interface. The kit also includes matrix keyboard, 7segment displays, lcd display connector, gpio pins etc.

 

features

* No need of IC programmer

* 7segment display

* Matrix keypad

* LCD interface

* Dip switches

* EEPROM 24c64

*RTC ds1307

* DB9 connector

* USB connector for power supply

* DPST relay

* Analog to digital converter ADC0804
